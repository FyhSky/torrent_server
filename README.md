一个BT实现的P2P点播后端服务
===========================

p2p下载服务, 可用于P2P播放器的后端.


##### 使用方法:

在编译后运行命令

```
torrent_server.exe girls-xvid-1080p-04.avi.torrent
```

然后使用vlc打开:

http://127.0.0.1:8889/girls-xvid-1080p-04.avi

即可享用p2p下载播放服务.
